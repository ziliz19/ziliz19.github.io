Details

I created this website as part of my Introduction to Database course. Users can use the website to record their income and expenses, and the site is capable of displaying charts to analyze transaction data, making it easy for users to track their spending. Users can also edit their personal information and transactions.
I developed the website using PHP, CSS, JavaScript, and SQL, allowing the system to record user interactions in an SQL database. I used open-source code from Google to visually present the data in charts. In addition to programming, I also researched design principles. I learned that a dark background can be more comfortable for users' eyes, and a clean interface makes it easier for users to read and navigate. With these principles in mind, I designed the website.
During the design process, I started by creating an Entity-Relationship Diagram (ERD) and sketching page layouts. Afterward, I implemented the website through programming. Throughout this project, I deepened my knowledge of PHP and successfully designed a complete website.
